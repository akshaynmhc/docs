  <Card
    title="1. What is WordPress, and what are its key features?"
  >
    WordPress is a free, open-source content management system (CMS) that allows users to create, manage, and publish digital content. Key features include a user-friendly interface, customizable themes, plugins, and a vast community of developers and users.
  </Card>
  <Card
    title="2. What is the difference between a theme and a plugin in WordPress?"
  >
    A theme controls the design and layout of a WordPress site, while a plugin adds functionality, extends features, or enhances the existing functionality.
  </Card>
